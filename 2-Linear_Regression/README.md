# 📈 Linear Regression - House Price Prediction

## 📌 Overview

This project focuses on predicting house prices using the Linear Regression algorithm.

The model learns the relationship between various features such as house size, number of bedrooms, and other factors to accurately estimate property prices.

---

## 🎯 Objective

- Predict continuous house prices  
- Understand linear relationships between features and target  
- Build a simple yet powerful regression model  

---

## ⚙️ Workflow

1. Data Collection & Cleaning  
2. Handling Missing Values  
3. Encoding Categorical Features  
4. Feature Scaling (if required)  
5. Model Training (Linear Regression)  
6. Model Evaluation  

---

## 📊 Model Performance

- **R² Score:** 0.99  
- **MAE:** 5,623  

---

## 📈 Metrics Used

- R² Score  
- Mean Absolute Error (MAE)  

---

## 💡 Key Insights

- Strong linear relationship between input features and house prices  
- Model achieved very high accuracy with minimal error  
- Simpler model outperformed more complex models in this case  
- Indicates well-structured and clean dataset  

---

## ⚠️ Note

Such high accuracy may indicate:
- Highly linear data  
- Or potential overfitting / data leakage  

Proper validation is recommended.

---

## 🛠️ Tech Stack

- Python 🐍  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

---

## 🙌 Conclusion

Linear Regression proved to be highly effective for this dataset, delivering excellent performance with minimal complexity. This project highlights the importance of selecting the right model based on data characteristics.
