# 🌳 Decision Tree Regressor - House Price Prediction

## 📌 Overview

This project focuses on predicting house prices using a Decision Tree Regressor.

The model learns decision rules from input features such as house size, number of bedrooms, location, and other attributes to estimate property prices.

---

## 🎯 Objective

- Predict continuous house prices  
- Capture non-linear relationships in data  
- Build an interpretable regression model  

---

## ⚙️ Workflow

1. Data Collection & Cleaning  
2. Handling Missing Values  
3. Encoding Categorical Features  
4. Model Training (Decision Tree Regressor)  
5. Model Evaluation  

---

## 📊 Dataset Features

### 🔢 Numerical Features
- Area (sqft)  
- Bedrooms  
- Age of House  

### 🔤 Categorical Features
- Location  
- Condition  
- Furnishing Status  

### 🎯 Target
- House Price  

---

## 📈 Model Performance

- **R² Score:** 0.95  
- **MAE:** 35,453  

---

## 📊 Metrics Used

- R² Score  
- Mean Absolute Error (MAE)  

---

## 💡 Key Insights

- Model captures complex non-linear relationships  
- Easy to interpret decision-making process  
- Sensitive to overfitting if not controlled  
- Performance improves with proper hyperparameter tuning  

---

## 🔧 Important Hyperparameters

- max_depth  
- min_samples_split  
- min_samples_leaf  

---

## ⚠️ Challenges

- Risk of overfitting  
- Requires tuning for optimal performance

The Decision Tree Regressor provides an interpretable and effective approach for predicting house prices. While it captures non-linear patterns well, careful tuning is required to achieve optimal performance.
