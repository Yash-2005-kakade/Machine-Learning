# 🎮 Esports Player Contract Prediction

## 📌 Overview

This project predicts whether an esports player should be offered a contract based on performance and behavioral attributes. A Random Forest Classifier is used along with GridSearchCV for hyperparameter tuning.

---

## 🎯 Problem Statement

Selecting the right esports player is a complex task involving multiple factors such as performance metrics and behavioral traits. This project provides a data-driven approach to assist in player selection.

---

## 🚀 Features

* Works with both numerical and categorical data
* Applies feature engineering to improve model performance
* Uses One-Hot Encoding for categorical variables
* Implements Random Forest Classifier
* Uses GridSearchCV for hyperparameter tuning
* Evaluates model using accuracy, precision, recall, and F1-score

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## ⚙️ Steps Performed

1. Data Loading
2. Handling Missing Values
3. Feature Engineering
4. Encoding Categorical Features
5. Train-Test Split
6. Model Training (Random Forest)
7. Hyperparameter Tuning (GridSearchCV)
8. Model Evaluation

---

## 📊 Results

* Accuracy: 86%
* Precision: 85%
* Recall: 85%
* F1 Score: 85%

---

## 🧠 Key Insights

* Data quality plays a major role in model performance
* Feature engineering significantly improves results
* Balanced metrics provide better evaluation than accuracy alone
