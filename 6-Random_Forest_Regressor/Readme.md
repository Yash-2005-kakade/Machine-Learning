# 📈 Content Creator Revenue Prediction

## 📌 Overview

This project predicts the monthly revenue of content creators based on performance, engagement, and platform-related factors. A Random Forest Regressor is used along with GridSearchCV for hyperparameter tuning to achieve high prediction accuracy.

---

## 🎯 Problem Statement

Estimating the revenue of content creators is complex due to multiple influencing factors such as audience engagement, content quality, and platform monetization. This project provides a data-driven approach to predict revenue more accurately.

---

## 🚀 Features

* Handles both numerical and categorical data
* Applies feature engineering to improve predictions
* Uses One-Hot Encoding for categorical variables
* Implements Random Forest Regressor
* Uses GridSearchCV for hyperparameter tuning
* Evaluates model using R² Score, MAE, and MSE

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## ⚙️ Steps Performed

1. Data Loading
2. Handling Missing Values
3. Feature Engineering
4. Encoding Categorical Features
5. Train-Test Split
6. Model Training (Random Forest Regressor)
7. Hyperparameter Tuning (GridSearchCV)
8. Model Evaluation

---

## 📊 Results

* R² Score: 0.86
* Mean Absolute Error (MAE): ~242
* Mean Squared Error (MSE): Low

---

## 🧠 Key Insights

* Revenue is strongly influenced by engagement metrics like retention and click rate
* Platform and audience region significantly impact earnings
* Feature engineering improves model performance significantly
