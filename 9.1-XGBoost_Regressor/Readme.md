# 🚀 XGBoost Regressor Project

## 📌 Overview
This project focuses on building a regression model using **XGBoost Regressor** and comparing its performance with Decision Tree and AdaBoost on the same dataset.

---

## 🎯 Objectives
- Build a regression model using XGBoost  
- Perform data preprocessing  
- Apply hyperparameter tuning  
- Evaluate model performance using MAE and R²  
- Compare with Decision Tree and AdaBoost  

---

## ⚙️ Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  

---

## 📊 Model Performance

### 🔹 Before Tuning
- MAE: 1.41  
- R² Score: 0.93  

### 🔹 After Tuning
- MAE: 1.22  
- R² Score: 0.95  

---

## ⚔️ Model Comparison

| Model            | MAE  | R² Score |
|------------------|------|--------|
| Decision Tree    | 2.32 | 0.82   |
| AdaBoost         | 1.58 | 0.91   |
| XGBoost (Tuned)  | 1.22 | 0.95 ✅ |

---

## 🔍 Key Features
- Used **XGBoost Regressor** for prediction  
- Applied **GridSearchCV** for hyperparameter tuning  
- Improved accuracy and reduced error significantly  
- Compared multiple models for better understanding  

---

## 🧠 Key Learnings
- XGBoost is a powerful boosting algorithm  
- Hyperparameter tuning improves performance  
- Ensemble methods outperform single models  

---

## 🏁 Conclusion
XGBoost outperformed Decision Tree and AdaBoost, achieving the highest accuracy with the lowest error after tuning.
